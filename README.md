# Recommandation_film

# 🎬 Projet Cinéma – Redynamisation & Système de recommandation

## ⚙️ Installation et exécution du projet

### 1. Cloner le projet
```bash
git clone https://github.com/unbiglow/Recommandation_cinema.git
cd Recommendation_cinema

### 2. Installer les dépendances
pip install -r requirements.txt

### 3.Lancer L'applicationn streamlit
streamlit run app.py OU
streamlit run dashboard.py
```
## 👥 Travail en équipe
Ce projet a été réalisé en équipe de 4 personnes dans le cadre d’une formation en Data analyst.

---

## 🎯 Contexte du projet
L’objectif est d’exploiter la data pour répondre à un double besoin :
1. Analyser le potentiel d’un cinéma situé en zone rurale (Creuse)
2. Développer une application interactive de recommandation de films basée sur les préférences utilisateurs

Le projet combine une approche **Business Intelligence (Power BI)** et une approche **Machine Learning / application interactive (Streamlit)**.

---

## ❓ Problématique
Comment utiliser la data pour aider à la redynamisation d’un cinéma en zone rurale et améliorer l’expérience utilisateur grâce à un système de recommandation de films ?

---

# 📊 Partie 1 – Dashboard Power BI (Analyse stratégique)

## 🎯 Objectif
Analyser le potentiel du cinéma afin d’optimiser la programmation et améliorer l’attractivité.

## 🧠 Analyse réalisée
- Étude de la zone géographique (Creuse)
- Analyse du marché cinématographique
- Identification des genres de films les plus populaires
- Analyse des tendances de fréquentation

## 📊 Dashboard Power BI
- Analyse territoriale
- Analyse des films et genres
- Indicateurs de performance
- Aide à la décision pour la programmation

---

# 🤖 Partie 2 – Application de recommandation (Streamlit)

## 🎯 Objectif
Développer une application interactive permettant de recommander des films en fonction des préférences utilisateurs.

## 🧠 Fonctionnement
- Système de recommandation basé sur similarité entre films
- Analyse des caractéristiques des films
- Génération de recommandations personnalisées
- Interface interactive via Streamlit

## 🧰 Technologies utilisées
- Python
- Streamlit
- Pandas
- Machine Learning (recommandation simple)

---

# 💡 Valeur métier du projet
Ce projet permet de :
- Transformer des données en décisions stratégiques
- Optimiser la programmation d’un cinéma
- Améliorer l’expérience utilisateur
- Proposer des recommandations de film pour l'utilisateur
